# 招聘简章：舆情管理专员

## 职位基本信息

- **岗位名称**：舆情管理专员
- **所属部门**：舆情监测部
- **汇报对象**：舆情部门主管
- **工作地点**：[北京]
- **薪资范围**：[请填写薪资范围，例如：6k-8k/月]

## 岗位职责

1. **数据清洗与校准**：
   - 负责对系统/数据库初筛后的汽车行业舆情数据进行人工二次清洗和复核，确保数据的准确性、完整性和相关性（去噪、去重、分类）。
   - 依据部门制定的数据清洗标准，对自动抓取的声量、情感倾向（正/负/中立）进行精准判定。
2. **舆情监测与预警**：
   - 针对服务的汽车客户进行日常网络舆情监测，及时发现潜在的负面信息或危机苗头，并第一时间上报。
3. **报告协助**：
   - 协助上级领导梳理舆情数据，制作日报、周报及月报。
   - 参与撰写基础的舆情分析报告，为客户提供数据支持。
4. **数据库维护**：
   - 协助维护舆情关键词库和黑名单库，不断优化系统抓取的精准度。

## 任职要求

1. **专业背景**：
   - 大专及以上学历，新闻学、传播学、统计学、信息管理或市场营销等相关专业优先。
2. **经验要求**：
   - 具备**舆情监测项目经验**或汽车行业相关工作经验者优先。
   - 有数据标注、信息处理相关实习或工作经验者加分。
3. **技能要求**：
   - **精通Excel**：熟练使用数据透视表等功能进行高效的数据处理和统计。
   - **加分项**：具备基础的数据库概念或SQL知识，能够更好地理解数据结构和清洗逻辑。
4. **综合素质**：
   - **细心负责**：对数据极度敏感，有耐心处理大量文本信息，具备高度的责任心（此岗位工作需极高的细致度，适合**细心、耐心、沉稳**的候选人）。
   - **抗压能力**：能适应突发舆情事件的紧急响应节奏。
   - 具备良好的文字功底和逻辑分析能力。

## 为什么加入我们？

- **业务稳定**：服务头部汽车品牌，业务量级大且稳定。
- **技术赋能**：公司正在全面推行AI化与数据库化转型，你将从繁琐的手工劳动中逐步解放，学习最前沿的"人+AI"协作工作流。
- **成长空间**：表现优秀者可晋升为高级舆情分析师或项目经理。
