# 抖音口播模板

## 模板说明

用于生成抖音口播类短视频脚本的提示词模板。
大脑使用此模板时，需要填充：{车型}、{卖点}、{参考爆款}、{用户评价}、{热梗}、{黑名单}、{切入角度}、{开头策略}

---

## 提示词模板

```
# 任务
为{车型}创作1条抖音口播短视频脚本。

# 参考爆款（学习结构和节奏）
以下是同类型的爆款脚本，学习它们的结构、节奏和说话方式，但不要抄袭内容：

{参考爆款}

# 真实用户声音（学习说话方式）
以下是真实车主对这款车的评价，用他们的语言风格和关注点来写：

{用户评价}

# 当前热梗（必须使用至少1个）
{热梗}

# 本次创作要求
- 车型：{车型}
- 核心卖点（只挑1-2个说）：{卖点}
- 切入角度：{切入角度}
- 开头策略：{开头策略}

# 切入角度说明
{角度说明}

# 开头策略说明
{开头策略说明}

# 格式要求
- 字数：250-350字
- 结构：hook（前3秒抓人）→ 铺垫 → 核心内容 → 收尾金句
- 用自然段，不要分点
- 口语化，像朋友聊天

# 禁止事项
- 禁止使用以下词汇：{黑名单}
- 禁止贬低其他品牌
- 禁止出现具体试驾场景描写
- 禁止使用"作为一个..."、"不得不说"等套话
- 禁止堆砌卖点，只说1-2个
- 禁止使用比喻和夸张修辞

# 输出
直接输出脚本正文，不要解释。
```

## 角度池

| 角度     | 说明                             | 适用场景           |
| -------- | -------------------------------- | ------------------ |
| 吐槽型   | "这车我本来不想推的..." 先抑后扬 | 网红车、争议车型   |
| 对比型   | "同价位开了5台，就这台..."       | 竞品多的价位段     |
| 场景型   | "老婆说再不换车就离婚..."        | 家用车、SUV        |
| 反常识型 | "所有人都说这车不行，但..."      | 冷门车、被低估车型 |
| 省钱型   | "穷人买车就看这几点..."          | 性价比车型         |
| 过来人型 | "买了3年说点真心话..."           | 老款车、经典车型   |

---

## 开头策略池

| 策略   | 示例                              | 效果                 |
| ------ | --------------------------------- | -------------------- |
| 提问   | "15万买SUV，你会看空间还是动力？" | 引发思考，提升互动   |
| 反常识 | "日系车省油？这台是个例外。"      | 制造冲突，吸引好奇   |
| 吐槽   | "这车唯一的缺点，就是太能装了。"  | 正话反说，降低广告感 |
| 热梗   | "{当前热梗}，这车就是..."         | 蹭热度，提升推荐     |
| 数字   | "6个月，3万公里，说说真实感受。"  | 建立可信度           |
