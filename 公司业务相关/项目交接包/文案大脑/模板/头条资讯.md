# 头条资讯模板

## 模板说明

用于生成今日头条资讯类稿件的提示词模板。
资讯类特点：字数更长、信息更全、标题党但正文要有料。

---

## 提示词模板

```
# 任务
为{车型}创作1篇今日头条资讯稿。

# 参考爆款标题（学习标题技巧）
{参考标题}

# 真实用户声音
{用户评价}

# 当前热点（可选结合）
{热梗}

# 本次创作要求
- 车型：{车型}
- 核心卖点：{卖点}
- 切入角度：{切入角度}

# 格式要求
输出三部分：

## 标题
三段式：主标题（吸引点击）| 副标题（核心信息）| 网友金句或感慨
示例：这车月销破2万 | CR-V到底强在哪 | 网友：闭眼买不会错

## 转发文案
- 不超过30字
- 必须包含车型名
- 后接5个话题词

话题词从以下随机选择：
#神仙颜值 #dou是好车 #自驾游好开车型 #城市通勤好车 #高保值车 #人生的第一辆车 #好车不贵经济实惠 #是时候买辆车了 #厉害了我的车 #每天推荐好车 #好车不等人 #带你懂车 #撸车日常 #说车聊车 #买车那点事儿 #家庭用车推荐

## 正文
- 字数：300-350字
- 自然段，不要分点
- 开头要有观点或思考，不要直接介绍产品
- 结尾要有总结性金句

# 禁止事项
- 禁止使用以下词汇：{黑名单}
- 禁止使用破折号和括号
- 禁止具体场景描述（试驾、驾驶）
- 禁止引用他人发言
- 禁止贬低其他品牌

# 输出
按"标题-转发文案-正文"顺序输出。
```

---

## 标题公式池

| 公式      | 结构                   | 示例                              |
| --------- | ---------------------- | --------------------------------- |
| 数据+悬念 | {数据}，{车型}凭什么？ | 月销2万台，CR-V凭什么？           |
| 反常识    | 都说{观点}，但{反转}   | 都说日系不安全，但这台...         |
| 对比      | {A}还是{B}？{结论}     | 轩逸还是卡罗拉？老司机这样选      |
| 身份+观点 | {身份}说：{观点}       | 开了5年的车主说：这车只有一个缺点 |
| 价格+价值 | {价格}，买到{价值}     | 15万，买到30万的享受              |

---

## 切入角度池

| 角度     | 说明                             |
| -------- | -------------------------------- |
| 销量解读 | 从销量数据切入，分析为什么卖得好 |
| 用户口碑 | 从真实用户评价切入               |
| 对比分析 | 和竞品对比，说清差异             |
| 使用场景 | 从用车场景切入，讲适合什么人     |
| 升级亮点 | 新款和老款对比，讲升级了什么     |
