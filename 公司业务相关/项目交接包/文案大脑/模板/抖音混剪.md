# 抖音混剪模板

## 模板说明

用于生成抖音二创混剪类短视频脚本的提示词模板。
混剪类特点：配合画面素材，文案更短、节奏更快、信息密度更高。

---

## 提示词模板

```
# 任务
为{车型}创作1条抖音混剪短视频脚本（配合画面素材使用）。

# 参考爆款（学习结构和节奏）
{参考爆款}

# 真实用户声音
{用户评价}

# 当前热梗（必须使用至少1个）
{热梗}

# 本次创作要求
- 车型：{车型}
- 核心卖点（只说1个）：{卖点}
- 切入角度：{切入角度}
- 开头策略：{开头策略}

# 混剪脚本格式
按镜头分段，每段对应一个画面，格式如下：

【镜头1-外观】{文案}
【镜头2-内饰】{文案}
【镜头3-细节】{文案}
...

# 要求
- 总字数：150-200字
- 每个镜头文案：15-30字
- 镜头数量：5-8个
- 节奏：快、短、有力
- 最后一个镜头必须是金句收尾

# 禁止事项
- 禁止使用以下词汇：{黑名单}
- 禁止长句（超过20字必须断句）
- 禁止堆砌卖点
- 禁止使用"首先、其次、最后"等过渡词

# 输出
直接输出脚本，按镜头分段。
```

---

## 角度池

| 角度       | 说明             | 示例开头                 |
| ---------- | ---------------- | ------------------------ |
| 快节奏测评 | 30秒说完一台车   | "30秒，说清楚这台车"     |
| 缺点盘点   | 先说缺点再说优点 | "这车3个缺点，想好再买"  |
| 买车清单   | 适合什么人买     | "3种人适合这台车"        |
| 对比结论   | 直接给结论       | "A还是B？直接告诉你答案" |
| 价格锚定   | 从价格切入       | "15万，能买到什么？"     |

---

## 镜头节奏参考

| 镜头    | 时长    | 内容                     |
| ------- | ------- | ------------------------ |
| 镜头1   | 2-3秒   | Hook，抓注意力           |
| 镜头2-5 | 各3-4秒 | 核心内容，一个镜头一个点 |
| 镜头6   | 3秒     | 金句收尾                 |
