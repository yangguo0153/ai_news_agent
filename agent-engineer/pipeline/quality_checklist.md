# Quality Checklist

## 通用
- [ ] 结论是否在前 3 句内说清？
- [ ] 是否引用了可追溯证据（日志/数据/反馈）？
- [ ] 是否有至少 1 条可执行建议？
- [ ] 是否给出明确 CTA？

## 公众号
- [ ] 标题是否表达明确收益/反差？
- [ ] 正文是否包含“过程 -> 决策 -> 结果 -> 复用方法”？
- [ ] 是否避免夸张承诺与无法验证表述？

## 抖音
- [ ] 3 秒内是否有钩子？
- [ ] 45-90 秒能否完整讲完一个观点？
- [ ] 是否含一个立刻可做的小动作？

## 记忆回写
- [ ] 当天日志是否已沉淀到 `memory/daily`？
- [ ] 新结论是否写入 `memory/MEMORY.md`（含证据+置信度）？
