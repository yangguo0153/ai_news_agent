# 抖音博主文案抓取任务

## 目标

模拟真实用户习惯，抓取抖音某个博主的前10条视频文案，并输出 Excel 文档。

## 技术方案

- 工具：agent-browser（已安装）
- 流程：打开博主主页 → 模拟滚动 → 抓取文案 → 输出 Excel

## 实现步骤

1. 用 `--headed` 模式打开浏览器（可视化调试）
2. 手动登录抖音（或加载已保存的登录状态）
3. 打开博主主页
4. 滚动加载视频列表
5. 提取每条视频的文案
6. 输出到 Excel

## 待确认

- [ ] 是否有抖音账号可登录
- [ ] 抓取视频标题还是详情页完整文案
