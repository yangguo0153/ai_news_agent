## 文件说明

### agent_swarm_guidelines.md
**Agent Swarm 工作规范文档**

定义了 5 个业务角色的工作规范和流程机制：
1. 客户经理 - 需求消化和确认
2. 策划者 - 传播方向和内容分配
3. Writer - 内容创作
4. 审核者 - 质量检查和循环控制
5. 输出校订者 - 合规校验和输出

包含：
- 架构概览（角色定义、Shared Context）
- 各角色的职责、流程、输出格式
- 流程控制机制（审核-修改循环）
- 分层披露策略
- 质量标准
- 异常处理

### settings.local.json
**本地配置文件**

Claude Code 的本地设置。
