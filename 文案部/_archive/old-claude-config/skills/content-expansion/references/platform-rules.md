# 平台规范

## 抖音

### 基本规格

| 项目     | 规格                   |
| -------- | ---------------------- |
| 脚本字数 | 250-350字              |
| 时长     | 60-90秒                |
| 标题格式 | 三段式竖屏（三行独立） |
| 发布文案 | 15-30字 + 话题标签     |

### 标题规范

- 必须三行，每行独立成句
- 必须含品牌+车型名（如：东风本田HR-V）
- 禁止出现竞品名称
- 核心逻辑：话题→观点→结论

### 发布文案规范

- 必须含品牌+车型名
- 必须体现Brief需求方向
- 禁止出现竞品名称
- 长度15-30字
- 末尾接话题标签（如 #HRV #东风本田）

### 内容风格

- 口语化，像真人说话
- 短句为主，每句不超过25字
- 有具体场景和细节
- 有观点，不平铺直叙

### 参考语料库

**必须阅读**：`02-参考学习/风格语料库/抖音专属/`

- `S1-犀利观点.md` — 制造冲突、反问句式
- `S2-干货科普.md` — 数据堆砌、专业术语
- `S3-幽默段子.md` — 拟人化、夸张表演
- `S4-生活观察.md` — 场景描写、细节刻画
- `S5-真实测评.md` — 体验流水账、具体数据

## 今日头条

### 基本规格

| 项目     | 规格          |
| -------- | ------------- |
| 稿件字数 | 650-800字     |
| 标题格式 | 单行，20-30字 |
| 配图     | 3-5张         |

### 参考语料库

**必须阅读**：`02-参考学习/风格语料库/今日头条专属/`

该目录下按风格分类的语料示例：

- `S1-犀利观点-头条语料.md` — 学习头条特有的展开方式
- `S2-干货科普-头条语料.md` — 学习数据论述的节奏
- `S3-幽默段子-头条语料.md` — 学习场景化长文写法
- `S4-生活观察-头条语料.md` — 学习细节描写和代入感
- `S5-真实测评-头条语料.md` — 学习体验记录的结构

**使用方式**：创作今日头条稿件前，先阅读对应风格的语料，学习其字数分配、段落节奏、展开逻辑。

### 标题规范

- 单行标题，20-30字
- 必须含品牌+车型名
- 可使用疑问句或陈述句
- 突出核心卖点或话题

### 内容结构

```
开头（50-80字）：场景引入 / 话题引入 / 问题引入
中段（300-500字）：核心内容，分2-3个自然段
结尾（50-100字）：总结观点 / 引导互动
```

### 内容风格

- 比抖音更详细，可展开论述
- 可引用具体数据和参数
- 段落分明，便于阅读
- 仍需口语化，避免说明文体

## 通用规范

### 必须遵守

- 无第一人称（我/我们/我的/咱们）
- 无结构化表述（XX方面/首先其次）
- 无广告腔（极致/震撼/硬核）
- 无拉踩竞品（只陈述本品优势）
- 时间逻辑合理（新车用"刚提车"视角）

### 信息来源

- 所有数据来自既定资料
- 不编造用户评价
- 不编造测试数据

### 话题标签

常用话题池（根据Brief调整）：

**品牌相关**：
#HRV #东风本田 #本田 #HRV向上为更大的世界 #CRV亲兄弟HRV上新

**场景相关**：#春节返乡 #家庭用车 #年轻人第一台车 #日常通勤

**功能相关**：
#SUV推荐 #十万级SUV #省油 #空间大

## 产出格式模板

### 抖音

```markdown
**方向X：[风格]+[角度]**

**标题**:
第一行（话题引入）
第二行（产品+卖点）
第三行（结论/数据/体验）

**脚本**:
[250-350字脚本内容]

**发布文案**: [15-30字] #话题1 #话题2 #话题3
```

### 今日头条

```markdown
**方向X：[风格]+[角度]**

**标题**: [20-30字单行标题]

**正文**:
[500-800字稿件内容，分段]

**配图建议**: [3-5张配图描述]
```

## Excel 终版输出格式

用户确认文案后，输出为 Excel 文件（.xlsx），不保留 CSV。

### 表头结构

| 序号 | 内容方向 | 发布平台 | 标题 | 正文 | 发布文案 |
| ---- | -------- | -------- | ---- | ---- | -------- |

### 字段填写规范

| 字段     | 说明                                 | 示例                           |
| -------- | ------------------------------------ | ------------------------------ |
| 序号     | 从1开始递增                          | 1, 2, 3...                     |
| 内容方向 | Brief中的话题方向代码                | T1新春购车指南、T2春节出行     |
| 发布平台 | 抖音 / 今日头条                      | 抖音                           |
| 标题     | 完整标题，抖音三段式用换行符(\n)分隔 | 春节返乡\n东风本田HR-V...\n... |
| 正文     | 完整脚本/稿件内容                    | [全文]                         |
| 发布文案 | 抖音：15-30字+标签；今日头条：填"无" | 春节返乡好搭档 #HRV #东风本田  |

### 文件命名

`04-产出仓库/[YYYYMMDD]-[车型]-[平台]-终版.xlsx`

示例：`04-产出仓库/20250206-HRV-抖音-终版.xlsx`
