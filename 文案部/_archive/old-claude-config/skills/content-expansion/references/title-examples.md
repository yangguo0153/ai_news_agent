# 抖音三段式标题库

## 格式规范

- 必须三行，竖屏显示
- 必须含品牌+车型名（如：东风本田HR-V）
- 核心逻辑：话题→观点→结论

---

## 第三段多样化模板

| 类型     | 模板                   | 示例                 |
| -------- | ---------------------- | -------------------- |
| 数据呈现 | [数据]确实[评价]       | 508L容积确实没吹牛   |
| 直接评价 | [卖点]确实[评价]       | 独立悬架确实不一样   |
| 场景结果 | [物品]随便[动作]       | 两大箱加行李箱随便塞 |
| 使用体验 | [身高]也能[动作]       | 一米八也能跷二郎腿   |
| 成本计算 | [成本]挺划算           | 一公里五毛多挺划算   |
| 反问引导 | 这[维度]是不是[评价]？ | 这价格是不是有点香？ |
| 车主反馈 | 车主：[短评]           | 车主：买早了也不亏   |
| 网友评价 | 网友：[短评]           | 网友：这底盘有点东西 |
| 行动号召 | [建议]再做决定         | 先去店里坐坐再说     |
| 对比结论 | [对比对象]差距明显     | 和扭力梁差距明显     |

**注意**：避免连续5篇使用"车主：XXX"或"网友：XXX"格式。

## 禁止的标题模式

### 禁止连续使用

- 连续3篇以上使用"车主：XXX"
- 连续3篇以上使用"网友：XXX"
- 连续3篇以上使用反问句结尾

### 禁止的表述

- 标题中出现竞品名称
- 使用绝对化用语（最、唯一、首个）
- 使用广告腔（极致、震撼、硬核）
