# Writer - 内容变量库：人设角度库

> 用途：提供不同人设的关注点、痛点和期望，Writer 根据人设选择合适的角度
> 原则：深入理解人设，创造真实的共鸣

---

## 一、宝妈视角

### 基本画像
- **年龄**：25-40岁
- **家庭状况**：有1-2个孩子，孩子年龄0-10岁
- **职业状况**：全职妈妈或职场妈妈
- **购车决策权**：高（家庭主要决策者之一）

### 关注点
1. **空间**：后备箱能装下婴儿车、玩具、尿不湿等
2. **安全**：主动安全系统、气囊数量、车身结构
3. **便利性**：上下车方便、座椅调节灵活、储物空间多
4. **舒适性**：后排空间大、座椅舒适、空调制冷快
5. **性价比**：养车成本低、保值率高

### 痛点
1. **空间焦虑**：
   - "以前的车，后备箱塞不下婴儿车"
   - "孩子的玩具、奶粉、尿不湿，一样都不能少"
   - "后排坐不下安全座椅，每次都要拆"

2. **安全担忧**：
   - "孩子在车里闹，我在前面急得团团转"
   - "高速上遇到突发情况，心里总是不踏实"
   - "担心孩子在车里不安全"

3. **便利性不足**：
   - "每次出门，光收拾孩子的东西就要半小时"
   - "孩子上下车不方便，每次都要抱"
   - "储物空间少，东西没地方放"

### 期望
- **省心**：不用担心空间不够、安全问题
- **安全**：孩子在车里安全、舒适
- **实用**：空间大、储物多、养车成本低

### 决策逻辑
1. 先看空间（后备箱、后排空间）
2. 再看安全（主动安全、被动安全）
3. 最后看性价比（养车成本、保值率）

## 二、孝子视角

### 基本画像
- **年龄**：30-45岁
- **家庭状况**：已婚，父母年龄60岁以上
- **职业状况**：职场中坚力量，收入稳定
- **购车决策权**：高（为父母购车或家庭用车）

### 关注点
1. **安全**：主动安全系统、被动安全配置、车身稳定性
2. **舒适**：后排空间大、座椅舒适、上下车方便
3. **面子**：品牌知名度、外观大气、配置齐全
4. **可靠性**：品质稳定、故障率低、保养便宜

### 痛点
1. **安全担忧**：
   - "父母年纪大了，反应慢，担心遇到突发情况"
   - "高速上遇到紧急情况，心里总是不踏实"
   - "担心父母开车不安全"

2. **舒适性不足**：
   - "以前的车，父母上下车要弯腰"
   - "后排空间小，父母坐着不舒服"
   - "父母说没事，但我知道他们坐得不舒服"

3. **面子问题**：
   - "给父母买车，不能太寒酸"
   - "朋友都给父母买了好车，我也不能落后"
   - "父母开出去，得有面子"

### 期望
- **安全可靠**：父母开车安全、坐车舒适
- **操作简单**：父母容易上手、不复杂
- **有面子**：品牌知名度高、配置齐全

### 决策逻辑
1. 先看安全（主动安全、被动安全）
2. 再看舒适（后排空间、座椅舒适）
3. 最后看面子（品牌、配置）

## 三、小夫妻视角

### 基本画像
- **年龄**：25-35岁
- **家庭状况**：新婚或恋爱中，暂无孩子或孩子很小
- **职业状况**：职场新人或中层，收入中等
- **购车决策权**：共同决策

### 关注点
1. **颜值**：外观时尚、内饰精致、有格调
2. **性能**：动力充沛、操控灵活、驾驶乐趣
3. **性价比**：价格合理、配置丰富、养车成本低
4. **实用性**：空间够用、油耗低、保值率高
5. **仪式感**：周末自驾游、情侣约会、生活品质

### 痛点
1. **预算有限**：
   - "想买好车，但预算有限"
   - "担心买了车，养车成本太高"
   - "朋友都有车了，我们还在挤地铁"

2. **选择困难**：
   - "车型太多，不知道选哪个"
   - "担心买错了，后悔"
   - "想要颜值，又想要性能，还想要性价比"

3. **生活品质**：
   - "想去哪儿都要考虑交通，太麻烦"
   - "周末想出去玩，但没车不方便"
   - "两个人的小日子，也要有仪式感"

### 期望
- **高性价比**：价格合理、配置丰富、养车成本低
- **有格调**：外观时尚、内饰精致、有品味
- **不后悔**：买了不后悔、开了不后悔

### 决策逻辑
1. 先看颜值（外观、内饰）
2. 再看性价比（价格、配置、养车成本）
3. 最后看实用性（空间、油耗、保值率）

## 四、职场精英视角

### 基本画像
- **年龄**：30-50岁
- **家庭状况**：已婚，有孩子或无孩子
- **职业状况**：企业高管、创业者、专业人士
- **购车决策权**：高（个人决策或家庭主要决策者）

### 关注点
1. **品质**：品牌知名度、做工精细、配置高端
2. **效率**：动力充沛、操控灵活、驾驶舒适
3. **面子**：外观大气、内饰豪华、有身份感
4. **可靠性**：品质稳定、故障率低、保养便宜
5. **智能化**：科技配置、辅助驾驶、车联网

### 痛点
1. **时间成本**：
   - "时间就是金钱，效率就是生命"
   - "堵车时，车内空间小，憋得慌"
   - "开车时，还要处理工作，需要舒适的环境"

2. **面子问题**：
   - "客户见面，车太寒酸，丢面子"
   - "朋友都开豪车，我也不能落后"
   - "车是身份的象征，得有品味"

3. **品质要求**：
   - "以前的车，小毛病太多，烦"
   - "养车成本高，不划算"
   - "想要品质好、可靠性高的车"

### 期望
- **高品质**：品牌知名度高、做工精细、配置高端
- **高效率**：动力充沛、操控灵活、驾驶舒适
- **有面子**：外观大气、内饰豪华、有身份感

### 决策逻辑
1. 先看品质（品牌、做工、配置）
2. 再看效率（动力、操控、驾驶舒适）
3. 最后看面子（外观、内饰、身份感）

## 五、使用原则

### 5.1 深入理解人设

- 不同人设有不同的关注点、痛点和期望
- 根据人设选择合适的角度和表达方式
- 创造真实的共鸣，而非生硬的套路

### 5.2 匹配卖点与人设

- 宝妈 + 空间 = 后备箱能装下婴儿车、玩具
- 孝子 + 安全 = 父母开车安全、坐车舒适
- 小夫妻 + 性价比 = 价格合理、配置丰富
- 职场精英 + 品质 = 品牌知名度高、做工精细

### 5.3 创造真实场景

- 基于人设的痛点，创造真实的场景
- 用具体的细节，而非抽象的描述
- 让读者产生"这说的就是我"的共鸣
