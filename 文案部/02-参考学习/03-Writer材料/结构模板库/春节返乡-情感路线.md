# Writer - 结构模板：春节返乡-情感路线

> 用途：Writer 创作时的结构框架（只有结构，无完整文案）
> 平台：抖音
> 字数：250-350字
> 调性：温和喜庆、场景化、情感共鸣

---

## 一、段落结构（4段式）

### 段1：春节场景唤醒（30-50字）

**必须包含**：
- 时间触发词（每到/一到/每年/每次）
- 具体画面（不是"春节到了"，而是"后备箱装满年货"）
- 情感铺垫（思念/期待/温暖）

**禁止**：
- ❌ 泛泛而谈（"春节马上到了"）
- ❌ 直接问候（"新年好"）
- ❌ 无画面感

**示例占位符**：
```
{时间触发词}春节，{具体画面}...{情感铺垫}
```

---

### 段2：以前vs现在对比（80-120字）

**必须包含**：
- 以前：大包小裹挤火车的具体细节（体感词）
- 现在：开车回家的具体场景（画面感）
- 对比：自然过渡，不生硬

**禁止**：
- ❌ 泛泛而谈（"走亲访友、采购年货"）
- ❌ 直接跳转到产品
- ❌ 缺少具体细节

**示例占位符**：
```
以前{具体痛点场景}...{体感词描述}
现在{开车回家场景}...{画面感描述}
```

---

### 段3：为什么选CR-V（80-120字）

**必须包含**：
- 大维度卖点（空间/安全/省心，选1个）
- 参数支撑（≤2个，融入场景）
- 自然过渡（不突兀）

**禁止**：
- ❌ 参数堆砌（"10气囊、ACE车身、980Mpa..."）
- ❌ 单一轻量配置（唱吧/LED大灯）
- ❌ 说明书式描述

**示例占位符**：
```
{自然过渡句}...CR-V的{大维度卖点}，{参数融入场景}
```

---

### 段4：情感升华（30-50字）

**必须包含**：
- 回到用户价值（不是产品功能）
- 温和喜庆的祝福
- 情感收束

**禁止**：
- ❌ 政策罗列（"现在购车正当时"）
- ❌ 直白描述（"CR-V是可靠的伙伴"）
- ❌ 激进表达

**示例占位符**：
```
{情感升华}...{温和祝福}
```

## 二、大维度卖点映射表

| 大维度 | 整合的参数 | 场景融入示例 |
|--------|-----------|-------------|
| **空间** | MM理念+后备箱容积+第二排空间 | "后备箱装满年货，第二排坐三个大人不挤" |
| **安全** | 10气囊+ACE车身+980Mpa防撞梁 | "高速上突然变道的车，系统提前预警" |
| **智能** | Honda SENSING 360+全套辅助 | "夜路上，大灯照得跟白天一样" |
| **省心** | 30年品质+全球1500万口碑+保值率 | "开了5年，换车时还能多回血一两万" |

**注意**：
- 不要单独讲"唱吧/LED大灯/单一声控"
- 参数必须融入场景，不能干巴巴罗列

## 三、创作流程

### Step 1：读取 Shared Context
```python
customer_brief = context.customer_brief
planner_brief = context.planner_brief
my_assignment = planner_brief.assignments[my_id]
```

### Step 2：随机抽取内容变量
```python
# 从内容变量库随机抽取
opening_sample = random.choice(opening_library)
detail_samples = random.sample(detail_library, k=3)
tone_samples = random.sample(tone_library, k=2)
```

### Step 3：填充结构模板
```python
# 按4段式结构填充
段1 = fill_opening(opening_sample, my_assignment.persona)
段2 = fill_contrast(detail_samples, my_assignment.scene)
段3 = fill_product(my_assignment.selling_point, detail_samples)
段4 = fill_closing(tone_samples)
```

### Step 4：自检
```python
# 检查禁用短语
if any(banned in content for banned in banned_phrases):
    regenerate()

# 检查字数
if not (250 <= len(content) <= 350):
```
