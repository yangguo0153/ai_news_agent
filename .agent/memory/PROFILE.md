## 沟通偏好
- 语言：中文。
- 风格：结论先行、少空话、可执行。
- 输出优先级：风险与结论 > 背景描述。
- 本地文件路径：禁止可点击 Markdown 本地链接；只接受“原文件名纯文本 + 单反引号绝对路径纯文本”。

## 协作约束
- 先计划再执行，复杂任务要文件化规划。
- 禁止用删报错/硬编码假数据掩盖问题。
- 关键结论要有证据并可回溯。
