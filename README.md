# 项目文档索引

本项目的所有关键文档均在此处列出，方便快速导航。

## 目录结构

- [公司业务文档](./公司业务/业务文档/INDEX.md) – 包含业务背景、计划及关键决策。
- [项目上下文](./公司业务/业务文档/PROJECT_CONTEXT.md) – 详细记录项目背景、关键决策和待办事项。

## 工作区结构约定

- `公司业务/`：公司相关文档与项目的归档区
- `公司业务/业务文档/`：业务文档与索引
- `公司业务/项目/`：公司项目（如 `media_dashboard`）
- `公司业务/部门/文案部`：指向根目录 `文案部` 的快捷入口
- `日常vibe/`：个人实验与临时项目
- `日常vibe/实验/`：可长期迭代的个人实验（如 `ai_news_agent`）
- `日常vibe/临时项目/`：短期或一次性项目
- `文案部/`：为便捷保留在根目录（与 `公司业务/部门/文案部` 同一入口）
